## Monty Hall Problem in PowerShell

This is a PowerShell implementation of the Monty Hall problem. It's summarized by this image taken from Wikipedia:

![Monty Hall Problem](https://raw.githubusercontent.com/jbrusoe/PowerShell/master/05MiscFiles/MontyHallProblem/ProblemStatement.JPG)

The question is then should you switch the door that was picked? The counterintuitive answer is that you should switch, and this file runs a demonstation of this scenario with and without switching. More information about it can be found here - https://en.wikipedia.org/wiki/Monty_Hall_problem.
