## Collatz Conjecture in PowerShell

This file demonstrates the Collatz Conjecture (or 3n + 1 problem) in PowerShell. This basically says to take any positive integer. If it's even, divide that number by 2. If it's odd, multiply it by 3 and add 1. The Collatz Conjecture states that by repeatedly following this pattern, you will eventually get back to 1.
