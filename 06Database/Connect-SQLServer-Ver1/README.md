## Connect to SQL Server with Invoke-SQLCmd

This file is a demonstration of how to connect and query data in a Microsoft SQL Server with [Invoke-SQLCmd](https://docs.microsoft.com/en-us/powershell/module/sqlserver/invoke-sqlcmd?view=sqlserver-ps). The data that I used for it (contained in the C02.csv file) was taken from - https://www.esrl.noaa.gov/gmd/ccgg/trends/.
