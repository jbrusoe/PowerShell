## Using PowerShell to Connect to Databases

These files demonstrate how to use PowerShell to connect to both a SQLServer and MySQL database. Two methods are used for each database type: [Invoke-SQLCmd](https://docs.microsoft.com/en-us/powershell/module/sqlserver/invoke-sqlcmd?view=sqlserver-ps) as well as the .NET [System.Data.SqlClient.SQLConnection](https://docs.microsoft.com/en-us/dotnet/api/system.data.sqlclient.sqlconnection?view=dotnet-plat-ext-5.0) class.
